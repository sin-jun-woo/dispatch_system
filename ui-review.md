# UI 리뷰 분석

## 관리자 대시보드 개선 포인트

### 1. 상단 요약 카드 (4개)
- 카드 배경색이 각각 다른 파스텔 톤 → 통일감 부족
- 카드 크기가 너무 크고 여백 과다
- "정산 완료 연산" 텍스트 불명확 → "정산 완료 금액"으로 변경

### 2. 기사 현황 배지 (대기중/운행중/수리중)
- 3개 배지가 별도 행 차지 → 요약 카드 아래 한 줄로 통합 가능
- 배지 크기 과대

### 3. 탭 네비게이션
- 탭 아이콘+텍스트 조합 → 텍스트만으로 충분
- 탭 우측 버튼(기사 등록 / 직접배차 생성)이 탭과 같은 줄 → 시각적 혼란

### 4. 필터 바
- 날짜 2개 + 기사 선택 + 배차 상태 + 필터 버튼 → 한 줄에 너무 많은 요소
- input 높이 불균일

### 5. 배차 목록 테이블
- "경로" 컬럼에 출발지/도착지/메모/도착기한이 모두 들어가 있어 복잡
- "생성일시" 컬럼에 생성일 + 승인일 2개 날짜가 겹쳐 있음
- 관리 컬럼 아이콘(✏/🚫/🗑) 간격 좁음

## 개선 방향
- 카드: 흰 배경 + 얇은 테두리 + 단색 아이콘으로 통일
- 필터: 컴팩트하게 한 줄 정리
- 테이블: 경로 컬럼 분리, 날짜 단순화
- 전체 배경: 현재 밝은 회색(oklch 0.97) 유지, 카드만 흰색
- 타이포: 제목 크기 줄이고 서브텍스트 색상 통일
